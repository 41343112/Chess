## 核心內容摘要

### 專題特色
本專題是一個使用 Qt/C++ 開發的完整國際象棋遊戲，具有以下特色：
- 完整的國際象棋規則實作
- 專業的類似 Chess.com 的介面
- 創新的使用者互動功能（拖放、音效、棋盤翻轉）
- 模組化的物件導向架構
- 跨平台支援（Windows、macOS、Linux）

### 技術亮點
- C++17 程式語言
- Qt 5.12+ / Qt 6.x 框架
- 完整的物件導向設計
- 三層移動驗證機制
- Qt Multimedia 音效整合
- 事件驅動的使用者介面

### 已實現功能
- ✓ 所有標準棋子移動規則
- ✓ 特殊走法（王車易位、吃過路兵、兵升變）
- ✓ 將軍、將死、逼和判定
- ✓ 點擊與拖放雙重移動模式
- ✓ 棋盤翻轉功能
- ✓ 移動音效回饋
- ✓ 完整的視覺回饋系統
