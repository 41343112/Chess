# UI Changes Visual Guide

## 開始對話框改進

### 改進前（舊版）

#### 難度選擇：
```
┌─────────────────────────────────┐
│ 難度： [▼ 中等          ]       │
│        簡單                      │
│        中等                      │
│        困難                      │
└─────────────────────────────────┘
```
- 使用下拉選單（QComboBox）
- 僅 3 個預設選項
- 難以微調難度

#### 對話框佈局：
```
┌────────────────────────────────────┐
│     歡迎來到象棋遊戲                │
│                                    │
│ ┌────────────────────────────────┐ │
│ │ 遊戲模式                        │ │
│ │ ○ 玩家對戰                      │ │
│ │ ○ 與電腦對戰                    │ │
│ └────────────────────────────────┘ │
│                                    │
│ ┌────────────────────────────────┐ │
│ │ 時間控制                        │ │
│ │ ...                            │ │
│ └────────────────────────────────┘ │
│                                    │
│ [      開始遊戲      ]             │
└────────────────────────────────────┘
```
- 固定大小
- 小螢幕可能無法完整顯示
- 無滾動支援

#### 難度選擇：
```
┌──────────────────────────────────────┐
│ 引擎難度等級：                        │
│                                      │
│ ├─────┼─────┼─────┼─────┼─────┤      │
│ 0     5     10    15    20          │
│      ╰──────●──────╯                 │
│                                      │
│     等級 10 - 中等                    │
│                                      │
│ 提示：0=極簡單，10=中等，20=極困難    │
└──────────────────────────────────────┘
```

- 21 個技能等級（0-20）
- 即時顯示當前等級和描述
- 刻度標記便於選擇
- 視覺化的難度選擇

## 功能對比表

| 功能 | 舊版 | 新版 | 改進 |
|------|------|------|------|
| 難度選項 | 3 個（簡單/中等/困難） | 21 個（0-20 技能等級） | ✅ 更精確 |
| AI 引擎 | 內建 Minimax | Stockfish UCI | ✅ 更強 |
| 難度調整 | 下拉選單 | 滑桿 | ✅ 更直觀 |
| 即時回饋 | 無 | 顯示等級描述 | ✅ 更友善 |
| 小螢幕支援 | 可能裁切 | 自動滾動 | ✅ 更靈活 |
| 開始按鈕 | 可能被擋住 | 始終可見 | ✅ 更易用 |

## 技術優勢

### 1. Stockfish 引擎
```
內建 AI                    Stockfish
等級 1-3    ───────→      等級 0-20
                          
弱 ─────────────────────── 強
```
- **棋力**：世界頂級引擎
- **可調整**：精確到單一等級
- **響應快**：1 秒內計算

### 2. 滑桿介面
```
下拉選單            滑桿
簡單 ─────→    ├─────●─────┤
中等 ─────→         │
困難 ─────→    視覺化，更直觀
```
- **視覺化**：一眼看出難度位置
- **流暢調整**：拖曳選擇
- **即時反饋**：顯示等級描述

### 3. 滾動支援
```
固定大小對話框          滾動對話框
┌─────────┐           ┌─────────┐
│ 標題    │           │ 標題    │ ↑
│ 內容 1  │           │ 內容 1  │ │ 滾動
│ 內容 2  │           │ 內容 2  │ │
│ 內容 3  │           │ 內容 3  │ ↓
└─────────┘           ├─────────┤
                      │ [開始]  │ ← 固定
                      └─────────┘
```
- **適應性**：支援任意大小螢幕
- **可擴展**：未來可輕鬆添加更多選項
- **用戶體驗**：按鈕始終可見

## 使用流程

### 選擇難度（新版）

1. **開啟對話框**
   - 啟動遊戲或點擊「新遊戲」
   - 選擇「與電腦對戰」

2. **調整難度**
   ```
   拖曳滑桿 → 看到即時等級 → 確認選擇
   ```
   - 拖動滑桿到想要的位置
   - 觀察標籤顯示的等級和描述
   - 找到合適的難度

3. **開始遊戲**
   - 選擇顏色（白/黑）
   - 設定時間控制（可選）
   - 點擊「開始遊戲」

### 難度建議

| 技能等級 | 描述 | 適合對象 |
|---------|------|---------|
| 0-3 | 極簡單 | 完全初學者 |
| 4-7 | 簡單 | 學習基本策略 |
| 8-12 | 中等 | 一般玩家 |
| 13-16 | 困難 | 有經驗玩家 |
| 17-20 | 極困難 | 進階玩家 |
